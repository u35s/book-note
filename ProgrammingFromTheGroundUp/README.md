## 深入理解程序设计(使用Linux汇编语言)

*  汇编指令

```
   在汇编程序中,任何以小数点(.)开始的指令都不会被直接翻译成指令
   这些针对汇编程序本身的指令,由于是有汇编程序处理,实际上并不会由计算机运行,
   因此被称为汇编指令或伪操作
```

*  寻址方式

```
直接寻址方式
movl 地址 %eax
索引寻址方式
movl 起始地址(,%索引寄存器,字长)
间接寻址方式(从寄存器指定的值加载值)
movl (%eax),%ebx
基址寻址方式与间接寻址类似,不同之处在与它将一个常量与寄存器中的地址相加
movl 4(%eax),%ebx 
立即寻址方式,直接将值加载到寄存器或指定位置
movl %12,%eax
寄存器寻址方式,仅仅是将数据移入或移出寄存器
```

* 栈
```
栈顶,内存底部,popl将值从栈顶弹出,该指令将值从栈顶移除,并把其放入寄存器或你选择的存储位置
我们不断将值入栈,栈会在内存中保持向下增长,直到达到存放代码或数据的地方
栈寄存器%esp总是包含一个指向当前栈顶的指针,无论栈顶在何处
每次用pushl将数据入栈,%esp所含的指针都会减去4,从而指向新的栈顶
如果想从栈中删除数据,只需用popl指令,该指令使%esp的值加4,并将先前栈顶的值放入你指定的寄存器
如果只是想访问栈顶的值,而不想移除,在间接寻址方式中%esp即可
```
